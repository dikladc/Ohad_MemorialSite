# אתר הנצחה - אוהד דותן

אתר זיכרון ל אוהד דותן ז"ל.

## הגדרת האתר

אתר זה נבנה כאתר סטטי ומתארח ב-Netlify (חינם).

### תכונות

- **דפי זיכרון**: 35 דפי הספד וזיכרונות
- **גלריות**: 6 גלריות תמונות
- **טופס זיכרונות חדש**: מאפשר למבקרים לשתף זיכרונות חדשים
- **הגנה מפני ספאם**: שימוש ב-Netlify Forms עם honeypot protection

### טכנולוגיות

- HTML/CSS/JavaScript
- Netlify Forms לטפסים
- Netlify Functions לפונקציות serverless
- UTF-8 encoding לתמיכה בעברית

### הוספת זיכרון חדש

מבקרים יכולים להוסיף זיכרונות דרך דף `/NewMemories.html`. הזיכרונות נשמרים ב-Netlify Forms ויכולים להיות מאושרים לפני הפרסום.

### פריסה

האתר מוכן לפריסה ב-Netlify או GitHub Pages. לפרטים נוספים, ראה את תיעוד Netlify.
